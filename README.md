# How to run
## Install dependencies
```
npm install
```

## Start the local development server (on port 8080)
```
npm start
```

## Ready for production?
## Build the production ready code to the /dist folder
```
npm run build
```

## Play your production ready game in the browser
```
npm run serve
```
